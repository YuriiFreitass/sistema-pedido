## 📌 Sobre o projeto

Projeto desenvolvido para praticar os conceitos de OOP em Java, simulando um sistema simples de pedidos com diferentes formas de pagamento.

---

## 🧠 Conceitos aplicados

| Conceito | Descrição |
|---|---|
| 📦 Classes e Objetos | Modelagem de entidades do domínio |
| 🔒 Encapsulamento | Controle de acesso aos atributos |
| 🧬 Herança | Reutilização e extensão de comportamentos |
| 🔌 Interfaces e Polimorfismo | Flexibilidade e desacoplamento |
| 🗂️ Enumerações (Enum) | Status e formas de pagamento |
| 🔗 Associação de objetos | Relacionamento entre entidades |

---

## ⚙️ Funcionalidades

- ✅ Cadastro de clientes e produtos
- ✅ Adição de itens ao pedido
- ✅ Cálculo automático do total
- ✅ Processamento de pagamento via **Pix**, **Cartão** ou **Boleto**
- ✅ Controle de status do pedido (`Aberto`, `Fechado`, `Cancelado`)

---

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/YuriiFreitass/sistema-pedido.git

# Acesse a pasta do projeto
cd sistema-pedido

# Compile e execute pelo seu IDE favorito (IntelliJ, Eclipse, VS Code)
```

> Requisito: **Java 17+**

# Sistema de Pedidos em Java

Sistema de pedidos desenvolvido em Java com foco em Orientação a Objetos.

## Sobre o projeto

Projeto desenvolvido para praticar os conceitos de OOP em Java, simulando
um sistema simples de pedidos com diferentes formas de pagamento.

## Conceitos aplicados

- Classes e Objetos
- Encapsulamento
- Herança
- Interfaces e Polimorfismo
- Enumerações (Enum)
- Associação de objetos

## Funcionalidades

- Cadastro de clientes e produtos
- Adição de itens ao pedido
- Cálculo automático do total
- Processamento de pagamento via Pix, Cartão ou Boleto
- Controle de status do pedido (Aberto, Fechado, Cancelado)

## Tecnologias

- Java 26
- IntelliJ IDEA

## Como executar

1. Clone o repositório
2. Abra o projeto no IntelliJ IDEA
3. Execute a classe `Main.java`
